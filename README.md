# Resume Screening System using NLP (Sentence Transformers)

An intelligent resume screening system that evaluates and ranks resumes against a job description using **semantic similarity** powered by **Sentence Transformers**.

This project simulates how modern Applicant Tracking Systems (ATS) work in real-world hiring pipelines.
